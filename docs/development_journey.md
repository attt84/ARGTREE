# 開発の軌跡と試行錯誤 (Development Journey)

本ドキュメントでは、`argdrasill` の開発において直面した課題と、それをどのように乗り越えて現在の形に至ったのか（試行錯誤の過程）を記録しています。

## 1. 議事録データの取り扱いにおける壁
**【初期の構想】**
当初は「会議全体のテキスト（数万文字）」をそのままLLM（Gemini）に投げ込み、会議全体のRAG（検索拡張生成）システムを構築しようとしていました。国会図書館の `/api/meeting` エンドポイントを使用し、会議1回分の全データセットを取得するアプローチです。

**【直面した課題】**
- **ノイズの多さ**: 議事録には本題と関係のない挨拶、議事進行の発言、別件の質問などが大量に含まれており、LLMが「本当に重要な議論」を見失う（ハルシネーションを起こす）ケースが多発しました。
- **コンテキストウィンドウの圧迫**: 会議全体のテキストは長大であり、Geminiのトークン制限を圧迫する上、処理時間（レイテンシ）が長くなる問題が発生しました。

**【解決策（ピボット）】**
「会議単位」ではなく**「発言（Speech）単位」**でのデータ取得へと方針を転換しました。`/api/speech` エンドポイントを使用し、ユーザーの検索キーワードを含む発言のみをピンポイントで抽出。これにより、情報密度が劇的に向上し、LLMの構造化精度とレスポンス速度が飛躍的に改善しました。

## 2. 議論の可視化：React Flowの導入
**【初期の構想】**
取得した議論を単なるテキストやMarkdownのリストとして表示することを考えていました。

**【直面した課題】**
単なる箇条書きでは、どの意見がどの意見に対する「反論」なのか、あるいは「解決策」なのかという**構造的・立体的な関係性**がユーザーに伝わらず、「議論の全体像を把握する」という目的が達成できませんでした。

**【解決策】**
フロントエンドに `React Flow (@xyflow/react)` を導入しました。
LLM（Gemini）に対して「テキストを返すのではなく、ノード（Node）とエッジ（Edge）で構成される厳密なJSONを返す」ようにプロンプトエンジニアリングを施し、バックエンドから直接グラフ構造データを返すアーキテクチャを採用しました。カスタムノードを作成し、「賛成（緑）」「反対（赤）」などの視覚的フィードバックを取り入れることで、ポートフォリオとしても見栄えのするUIが完成しました。

## 3. RAGの深化：一次情報（ソース）への回帰
**【初期の構想】**
ツリーを生成しただけで満足し、各ノードは「短い要約文」を表示するのみにとどまっていました。

**【直面した課題】**
「短い要約だけでは結局何も学べない（ファクトチェックができない）」という重大な欠陥に気づきました。LLMが要約したテキストが本当に国会で発言されたものなのか、ユーザーが確かめる術がありませんでした。

**【解決策】**
**「論点の深掘り（Node Detail）」機能**を実装しました。
ノードをクリックした際、そのノードのラベルと背後にある議事録データを再度Geminiに渡し、「なぜこの論点が挙がったのか」「具体的な発言の引用」を出力させるようにしました。さらに、国会APIから取得した `speechURL`（公式の議事録リンク）をプロンプトに動的に埋め込み、**「AIの解説の中に必ず一次情報へのリンクを含める」**よう強制することで、学習ツール・調査ツールとしての信頼性を担保しました。

## 4. 限界の突破：超検索（Query Expansion）の誕生
**【直面した課題】**
国会図書館APIは仕様上「OR検索」をサポートしておらず、複数キーワードを入力するとすべて「AND検索」になってしまうという制約がありました。例えば「AI規制」で検索すると「人工知能のガバナンス」という重要な発言を取りこぼしてしまいます。

**【解決策】**
**「LLMによるクエリ拡張（Query Expansion）」**を独自実装しました。
ユーザーが「超検索」を有効にした場合、バックエンドで以下のアプローチをとります。
1. Geminiにユーザーの検索意図を解釈させ、関連キーワードを複数生成させる（例：「AI規制」「人工知能 ガバナンス」「ディープフェイク」）。
2. 生成された複数のキーワードを使って、バックエンドから国会APIへ**並列で複数のリクエスト**を送信。
3. 取得したデータをマージして一つのコンテキストとしてLLMに渡す。

これにより、APIの仕様上の制約をソフトウェアアーキテクチャの工夫で乗り越え、セマンティック検索（意味検索）に近いハイブリッドなRAG体験を実現しました。
